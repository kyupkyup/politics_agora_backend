# 당선 정치인 공약 열람 앱
정치판은 당선된 정치인의 공약을 열람할 수 있는 앱입니다.

## 제작자
안드로이드 : 이경엽

백엔드 : 이경엽

## 목적
기술을 이용해 사람들에게 선한 영향력을 끼칠 수 있는 방법으로 조금 더 바른 투표를 도울 수 있는 앱을 만듬

## 다운로드
https://play.google.com/store/apps/details?id=com.kyupkyup.politicsagora

## 기능
1. 문재인 대통령 공약 열람

2. 시·도지사 공약 열람

3. 기타 정치인(정당 국회의원) 공약 열람
